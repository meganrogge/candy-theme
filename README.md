# Candy Theme

A theme inspired by candy for VS Code.

</div>

## Installation

1. Install theme from the [Marketplace](https://marketplace.visualstudio.com/items?itemName=meganrogge.candy-theme)
2. Go to `File > Preferences > Color Theme`
3. Select `Candy`
